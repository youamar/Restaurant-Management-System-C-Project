# 🍽️ Restaurant Management System

This is an Object-Oriented Programming (IOOP) project developed in C#. It aims to provide an intuitive user interface and a foundational operational framework for a restaurant management system.

## 🌟 Features
* **Main Page Navigation**: The core entry point of the system, providing seamless navigation to various management functions.
* **Hall & Seating Management**: Visually displays the layout and status of the restaurant's hall and dining tables, supported by graphical assets.
* **Menu Management**: Includes the categorization of restaurant dishes, detailed information display, and food/beverage image management.

## 💻 Technologies Used
* **Programming Language**: C#
* **Development Environment**: Visual Studio 2022
* **Core Concepts**: Object-Oriented Programming (OOP) principles such as Classes, Encapsulation, and Inheritance.

## 📂 Project Structure
* `Main Page/` - Contains the primary Windows Forms / UI source code and backend logic.
* `Hall Picture/` - Stores image resources related to the restaurant hall and table status.
* `Menu Picture/` - Stores image resources for food and beverages shown in the menu.
* `Other Picture/` - Stores additional icons and supporting UI graphics required by the system.
* `Main Page.sln` - The Visual Studio solution file to load the entire project with a single click.
